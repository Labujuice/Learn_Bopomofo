# BoPoMoFo ㄅㄆㄇㄈ 發音出題機

## Description and purpose 
- 這是一個幫助幼稚園、小學生學習 注音符號 ㄅㄆㄇㄈ 的出題機。
- 主要是協助爸爸媽媽、哥哥姊姊、老師 可以輕鬆幫小朋友出題，讓小朋友練習發音。
- 具備語音合成 (TTS) 發音輔助、音效與童趣視覺，提供有趣的學習體驗。

## Requirement List

### 1. 遊戲關卡與出題規則
- **遊戲分成四個不同的階級**：
  - **Lv.1 🌟 符號召喚師**：認讀 37 個基礎符號。出題為單一注音符號，並可隨機搭配聲調提示（如：請唸「ㄇˇ」三聲）。
  - **Lv.2 ⚔️ 雙音雙打組**：兩拼組合（聲母 + 韻母，或介音 + 韻母）。如「ㄅㄚ」、「ㄧㄢ」，需搭配隨機聲調。
  - **Lv.3 🏰 三音大合奏**：三拼組合（聲母 + 介音 + 韻母）。如「ㄅㄧㄢ」，需搭配隨機聲調。
  - **Lv.4 🏆 終極解碼王**：隨機混合兩拼與三拼的題目，需搭配隨機聲調。
- **⚠️ 拼音合法性過濾**：系統必須過濾不合法的注音組合（例如不能出現「ㄐㄞ」、「ㄅㄨㄥ」等無法發音的拼音），需內建合法拼音資料庫或拼音規則。
- **題數與結束條件**：
  - 每輪遊戲固定可選擇 **10 題**、**20 題**、**30 題**。
  - 答錯 **3 次**（扣完 3 顆愛心）遊戲會提前結束。
  - 若答對全部 10 題或在 3 次生命內完成，算通關，並獲得額外通關加分（如 10 分）。

### 2. 計分與排行榜
- **計分方式**：
  - 答對一個注音符號得一分。Lv.1 一題 1 分；Lv.2 一題 2 分；Lv.3 一題 3 分；Lv.4 根據出題的注音符號字數計分。
  - 通關（完成 10 題且生命值 > 0）可額外獲得 **+10 分通關獎勵**。
- **排行榜系統**：
  - 記錄：排名、姓名、分數、關卡等級、日期與時間。
  - 排行榜需提供 **「混合顯示」** 與 **「依等級分類（Lv.1 ~ Lv.4 獨立頁籤）」** 顯示功能，以求公平性。

### 3. UI Flow 與頁面架構

- **主選單 (Main Menu)**：
  - **姓名輸入框**：若未輸入，系統隨機產生「可愛形容詞 + 水果」名字（如：閃亮的水蜜桃、聰明的蘋果）。
  - **Level 1 ~ 4 選擇按鈕**。
  - **注音符號表 (Symbol Table)**：
    - 直接展示所有注音符號（聲母、介音、韻母）。
    - **互動發音**：點擊任何一個注音符號，會透過語音合成 (TTS) 播放該符號的正確發音。
  - **排行榜按鈕**。
  - **設定按鈕**：
    - 背景音樂 (BGM) 音量調節：0% ~ 100%。
    - 清除排行榜資料按鈕。

- **遊戲進行畫面 (Game Screen)**：
  - **上方資訊欄**：顯示參賽者姓名、目前分數、剩餘生命值（以 3 顆紅色愛心動畫表示）。
  - **中間出題區 (佔 70% 空間)**：
    - 大字體顯示目前的注音題目（加註聲調標誌）。
    - **🔊 聽看看按鈕**：家長或小朋友可點擊此按鈕，透過語音合成 (TTS) 聆聽該題目的正確發音。
    - 答對/答錯時，畫面中間會彈出可愛的綠色圈圈或紅色叉叉動畫。
  - **下方評判按鈕**：
    - 圈圈 (答對) 與 叉叉 (答錯) 按鈕，供主持人（家長/老師）操作。
    - 按下時播放對應音效（叮咚代表答對，benben代表答錯）。

- **遊戲結算畫面 (Result Screen)**：
  - 顯示本局姓名、最終分數、關卡等級、遊玩日期時間、答對率。
  - 顯示「🎉 恭喜通關」或「😢 繼續加油」等回饋。
  - 提供「再玩一次」與「回主選單」按鈕。

### 4. 系統與音效實作細節
- **音效系統**：
  - 使用 Web Audio API 即時合成「叮咚」與「benben」音效，免除外部檔案載入問題。
  - 內建輕快、兒童風的背景音樂（BGM），需符合瀏覽器 Autoplay 政策。
- **語音合成 (TTS)**：
  - 使用 Web Speech API (`SpeechSynthesis`) 進行注音發音輔助。
- **資料持久化**：
  - 排行榜記錄與音量設定需儲存在 `localStorage` 中。
- **響應式網頁設計 (RWD)**：
  - 介面需針對手機與平板進行優化，適合家長單手操作評判。
