# BoPoMoFo ㄅㄆㄇㄈ 發音出題機

## Description and purpose 
- 這是一個幫助幼稚園、小學生學習 注音符號 ㄅㄆㄇㄈ 的出題機
- 主要是協助爸爸媽媽、哥哥姊姊、老師 可以輕鬆幫小朋友出題, 讓小朋友可練習發音

## Requirement List
- [ ] 遊戲分成四個不同的階級：

    | 等級 | 關卡名稱     | 核心學習內容（對應您的想法）    | 闖關概念描述                         |
    | ---- | ------------ | ------------------------------- | ------------------------------------ |
    | Lv.1 | 🌟 符號召喚師 | 認讀37個基礎符號與聲調          | 收集所有的注音音符精靈。             |
    | Lv.2 | ⚔️ 雙音雙打組 | 兩拼（聲+韻）與 結合韻（介+韻） | 讓兩個音符牽牽手，一起發出新聲音。   |
    | Lv.3 | 🏰 三音大合奏 | 三拼（聲+介+韻）的完整組合      | 挑戰最強的三拼合體技，解鎖所有漢字。 |
    | Lv.4 | 🏆 終極解碼王 | 隨機混合的兩拼與三拼            | 混搭都可以念出，成為注音大師。       |

- [ ] 階級可以在遊戲開始前就讓使用者選擇, 如果答錯三次遊戲就會結束回到主畫面
- [ ] 遊戲計分方式為 一個注音符號答對一分, 也就是說 lv1 一題一分, lv2 一題二分, lv3 一題三分, lv4 則是根據出題有幾個注音符號就幾分
- [ ] UI Flow 則以簡單為主

  - 主選單
    - 輸入名字的框框, 如沒輸入則為隨機水果名字
    - Level 1
    - Level 2
    - Level 3
    - Level 4
    - 注音符號表(進去後一個頁面直接顯示出來)
      - 聲母：
        - ㄅ ㄆ ㄇ ㄈ ㄉ ㄊ ㄋ ㄌ ㄍ ㄎ ㄏ ㄐ ㄑ ㄒ ㄓ ㄔ ㄕ ㄖ ㄗ ㄘ ㄙ
      - 介音：
        - ㄧ ㄨ ㄩ
      - 韻母：
        - ㄚ ㄛ ㄝ ㄞ ㄟ ㄠ ㄡ ㄢ ㄣ ㄤ ㄥ ㄦ
    - 排行榜
      - 排名
      - 姓名
      - 分數與等級
      - 日期與時間
      - 可以選擇混合顯示或者依據等級分類顯示
    - 設定
      - 背景聲音: 0% ~ 100%
      - 清除排行榜
  
  - Level1 ~ Level 4 各關卡內容
    - 出題顯示界面(佔據 70% 空間, 畫面正中間)
    - 最上方顯示目前的參賽者名字、分數、還有多少命(紅色愛心表示)
    - 最下方為 圈圈與叉叉
    - 當遊戲主持人按下圈圈叉叉時, 畫面中間會顯示 圈與叉, 並且播放音效(叮咚 for 圈圈; benben for 叉叉)
    - 當累積三個叉叉之後, 命就沒有了, 遊戲結束
    - 遊戲結束後 顯示名字、分數、日期時間、Level
